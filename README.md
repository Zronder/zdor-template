# 工程配置效率提升说明

-   快捷键说明

| 快捷键                | 说明                              |
| --------------------- | --------------------------------- |
| option + shift + F    | 快速格式化代码                    |
| command + control + T | 添加函数注释                      |
| command + control + I | 添加文件注释                      |
| option + shift + O    | 自动删除头部不必要引用            |
| ii                    | 快速创建 react 函数组件及相关引用 |
| log                   | 快速输出 console.log              |
